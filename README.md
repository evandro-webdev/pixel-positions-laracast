# 📚 Projeto de Estudos Laracasts

Este é um projeto de estudos baseado nos tutoriais do Laracasts, focado em aprender Laravel, Tailwind CSS e outras tecnologias modernas de desenvolvimento web.

## 🛠️ Stack Tecnológica

- **Backend**
  - Laravel 10.x
  - PHP 8.2+
  
- **Frontend**
  - Tailwind CSS 4.x
  - Blade

- **Banco de Dados**
  - SQLite
